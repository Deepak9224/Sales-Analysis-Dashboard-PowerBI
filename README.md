Sales Analytics Dashboard | Power BI

## Project Overview

This project analyzes retail sales performance using the Superstore dataset.

The dashboard provides insights into:

- Sales Performance
- Profitability
- Regional Analysis
- Product Analysis
- Customer Segments
- Time-Based Trends

---

## Dashboard Pages

### Executive Summary
- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Profit Margin
- Category Analysis
- Segment Analysis

### Regional Analysis
- Sales by Region
- Profit by Region
- Sales by State
- Profit by State

### Product Analysis
- Top 10 Products by Sales
- Top 10 Products by Profit
- Top 10 Loss-Making Products
- Discount vs Profit Analysis

### Time Analysis
- Sales Trend Analysis
- Profit Trend Analysis
- Sales vs Profit Comparison

---

## Key Insights

- Technology is the highest-selling category.
- Consumer segment contributes the most revenue.
- West region generates the highest sales and profit.
- Several products generate negative profit despite generating sales.
- Sales and profit peaked in 2017.
